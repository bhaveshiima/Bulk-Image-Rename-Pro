Step 1 — Upload Your Images

Drag and drop image files directly onto the upload zone on the left sidebar, or click the zone to open your file picker. You can select as many files as you need in one go — they will appear instantly as thumbnail cards, sorted alphabetically.

Supports JPG, PNG, WEBP, GIF and all common image formats. Everything runs entirely in your browser — no files are ever sent to a server.

Step 2 — Export List

Click the Export List button in the left sidebar. It will download an Excel file (ImageList.xlsx) with two columns — Original_Filename which already contains all your image filenames, and New_Filename which is left blank for you to fill in. Open the file in Excel and type the desired new name for each image in the New_Filename column, then save the file.

Do not change the Original_Filename column — it is used to match each row back to the correct image when you import.

Step 3 — Import List

After filling in the New_Filename column in Excel and saving the file, come back here and click the Import List button. Select the saved Excel file — the tool will instantly read the new names and apply them to the matching images on screen. You will see all the thumbnails update with their new names right away.

Names are matched by Original_Filename, so even if you reorder or sort rows in Excel, the correct name will always go to the correct image.

Step 4 — Download ZIP

After importing, click the Download ZIP button in the top-right of the preview panel. All your images will be packaged into a single file called PatronPhotos.zip and downloaded to your device. Extract the ZIP and all photos will have their new names. Enjoy! 🎉

The original file format is always preserved — a .jpg stays a .jpg regardless of the new name you give it.
Note: Original file extensions are strictly preserved regardless of the name you type. The Download ZIP button is disabled until all images have unique filenames.
